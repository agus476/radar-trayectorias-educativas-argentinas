# Guion de charla: Analítica de Riesgo de Abandono Escolar

## 1. Apertura

El abandono escolar muchas veces se detecta tarde. La pregunta que guía este proyecto es si podemos anticipar perfiles de riesgo usando datos educativos públicos.

## 2. Problema

El sistema educativo genera datos todos los años, pero esos datos suelen quedar fragmentados en archivos administrativos. El desafío es transformarlos en señales accionables.

## 3. Dataset

El proyecto integra información de matrícula, trayectoria por sexo e infraestructura institucional para construir una mirada longitudinal y multivariable.

## 4. Pipeline

El flujo técnico incluye formateo, limpieza, tratamiento de nulos/outliers, ingesta SQL y visualización interactiva.

## 5. Dashboard

Presentar las cinco dimensiones:

1. Tendencia histórica.
2. Flujo de retención y brecha de promoción.
3. Sobreedad y abandono.
4. Infraestructura escolar.
5. Perfil de riesgo.

## 6. Insight central

El perfil estatal, rural y sin conectividad concentra señales de riesgo y permite pensar en una priorización de intervenciones.

## 7. Machine Learning futuro

El dashboard es la base exploratoria para un modelo de clasificación de riesgo que permita detectar instituciones vulnerables antes de que el abandono se consolide.

## 8. Cierre

El proyecto muestra cómo pasar de datos educativos históricos a una herramienta de decisión para prevención y focalización de recursos.
